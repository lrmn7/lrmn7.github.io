## Running Locally

Clone the project

```bash
git clone https://github.com/lrmn7/dev-blog.git
```

Go to the project directory

```bash
cd dev-blog
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
